# projeto-ia-reconhecimento
Aplicação interativa desenvolvida em Python que utiliza Inteligência Artificial para rastrear e analisar movimentos corporais, faciais e gestos em tempo real. O projeto integra o MediaPipe Holistic para o processamento de visão computacional e o Streamlit para uma interface de usuário moderna e responsiva, estando totalmente preparado para deploy em nuvem (Render).
